# BWDK Swift SDK — Integration Guide for Agents

You are integrating **BWDK (Buy With DigiKala)** into a Swift server project (Vapor / Perfect / Kitura) via this SDK. Read this file **first**, then consult the companion references below.

## BWDK constants

- **Host:** `https://bwdk-backend.digify.shop`
- **Auth scheme:** `MerchantAPIKeyAuth` — header `Authorization: <api_key>`.
- **Main API class:** `DefaultAPI` (note the fully-capitalized `API` — Swift generator's convention).
- **Distribution:** Carthage or CocoaPods (see `README.md`). Swift Package Manager is not generated by default for this template.

## Minimal wrapper pattern

```swift
import OpenAPIClient

OpenAPIClientAPI.basePath = "https://bwdk-backend.digify.shop"
OpenAPIClientAPI.customHeaders["Authorization"] = ProcessInfo.processInfo.environment["BWDK_API_KEY"] ?? ""

DefaultAPI.orderApiV1CreateOrderCreate(orderCreate: payload) { order, error in
    guard let order = order, error == nil else { return /* handle error */ }
    // use order
}
```

If async/await is enabled in your toolchain, prefer the generated `async` variants.

Method names are camelCase and OpenAPI-generated (e.g. `orderApiV1CreateOrderCreate`, `orderApiV1ManagerVerifyCreate`). Look them up in `docs/DefaultAPI.md`; do **not** guess.

## Integration invariants

1. **SDK only.** Never call BWDK with `URLSession` directly or `Alamofire` for BWDK endpoints.
2. **Callback flow.** After payment, BWDK redirects the customer to your `callback_url`. Load the order (`DefaultAPI.orderApiV1ManagerRetrieve`), switch on `order.status` per `FLOWCHART.md`, then call `DefaultAPI.orderApiV1ManagerVerifyCreate` — `verify` is mandatory before `SHIPPED`.
3. **Errors.** Completion handlers deliver `(T?, Error?)`; cast the error to `ErrorResponse` to see HTTP status and body. Retry only on transport errors, never on 4xx.
4. **Pinning.** Pin a concrete `vX.Y.Z` in your `Cartfile` / `Podfile`; never track a branch.

## Project conventions

- **Vapor:** wrap SDK calls in a service registered on the `Application` via `app.storage`; do not initialize the SDK per request.
- **Thread safety:** `OpenAPIClientAPI` globals are set once at boot; do not mutate them from request handlers.
- **Codable models:** all models conform to `Codable` — use them directly in your own JSON responses.
